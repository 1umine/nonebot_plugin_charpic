## 版本

v0.0.1

⚠ 适配nonebot2-2.0.0beta.1；

## 安装

1. 通过`pip`或`nb`安装；

命令

`nb plugin install nonebot_plugin_charpic`

`pip install nonebot_plugin_charpic`

## 功能

合成字符画(或 gif )

## 命令

`字符画` + `图片`

⚠ 需要 nonebot2 配置的命令前缀，如果没配置默认 `/` （好像是吧）
